# TreeScan
🌳 TreeScan: Hệ thống Quét và Phân tích Cây Xanh Từ Xa
TreeScan là một phần mềm/hệ thống ứng dụng Trí tuệ Nhân tạo (AI) và Thị giác máy tính (Computer Vision) để phân tích cây xanh mà không cần tiếp xúc trực tiếp. Hệ thống hỗ trợ nhận diện tự động, trích xuất thông số vật lý và phân loại giống cây với độ chính xác cao.

✨ Tính năng nổi bật (Key Features)
🔍 Nhận diện giống cây (Species Identification): Tự động phân tích đặc điểm của lá, thân và vỏ để xác định chính xác giống cây.
📏 Đo lường kích thước (Dimension Measurement): Trích xuất thông tin chiều cao tổng thể và độ rộng tán cây từ hình ảnh/video đầu vào.
🌳 Phân tích hình dáng (Shape Recognition): Nhận diện và phác thảo hình dáng tổng quan của cây, hỗ trợ cho việc quản lý cảnh quan và lâm nghiệp.
📸 Quét không tiếp xúc (Remote Scanning): Thu thập dữ liệu từ xa thông qua camera mà không cần các thiết bị đo lường thủ công.
💻 Công nghệ sử dụng (Tech Stack)
Ngôn ngữ lập trình: Python

Computer Vision & Deep Learning: YOLOv8 (phát hiện đối tượng và đo lường bounding box), MobileNetV2 / ONNX (phân loại hình ảnh học sâu).
Xử lý dữ liệu: OpenCV, NumPy.

🌳 TreeScan: Remote Tree Scanning and Analysis System
TreeScan is an AI-powered software utilizing Computer Vision to analyze trees without direct physical contact. The system supports automatic recognition, physical parameter extraction, and high-accuracy species classification.

✨ Key Features
🔍 Species Identification: Automatically analyzes leaf, trunk, and bark characteristics to accurately determine the tree species.
📏 Dimension Measurement: Extracts overall height and canopy width information from input images or video feeds.
🌳 Shape Recognition: Identifies and outlines the overall shape of the tree, assisting in landscape and forestry management.
📸 Remote Scanning: Collects data remotely via cameras, eliminating the need for manual measurement tools.
💻 Tech Stack
Programming Language: Python

Computer Vision & Deep Learning: YOLOv8 (for object detection and bounding box measurement), MobileNetV2 / ONNX (for deep learning image classification).
Data Processing: OpenCV, NumPy.
